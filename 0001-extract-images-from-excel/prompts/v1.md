在 main.py 中实现主要逻辑：提取 excel 的图片，以及图片链接中的图片
按一定顺序写入到 excel 的同目录下，同目录新建文件夹，与 excel 同名，图片放在里面

需要有适当的界面，用 PySide6，注意需要适配，windows7，windows10， windows11， windows11 arm，mac，mac m1 等系统，
所以需要让我可以很方便的打包，用 Nuitka 打包

后续我会经常需要打包，界面，可以把这些抽成公共的类库

省钱小技巧（尤其是用 matrix 时）

用 fail-fast: true（默认 true）：一个组合失败就停掉其他，节省分钟。
优先用 ubuntu-latest（最便宜，1x 倍率），避免 macOS（10x 倍率）。
把小 job 合并成一个（减少 job 数量，因为每个 job 至少算 1 分钟）。
用缓存（actions/cache）加速依赖安装，缩短 job 时间。
公共 repo → 完全免费，考虑把工具开源。
监控使用量：在 repo Settings → Billing → GitHub Actions 查看消耗。

使用 github actions matrix 来打包发布各个版本，暂时只需要 mac 和 windows11 版本
